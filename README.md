This repository contains my notes, assignments, and practice implementations 
from the NPTEL course **"Introduction to Machine Learning"** by 
**Prof. Balaraman Ravindran, IIT Madras**.

## Course Details
- Platform: NPTEL / Swayam
- Instructor: Prof. Balaraman Ravindran, IIT Madras
- Duration: 12 weeks
- Start Date: 20 July 2026

## Contents
- `notes/` — Week-wise summarized notes and key concepts
- `assignments/` — Solutions to weekly assignments/quizzes
- `practice/` — Hands-on implementations of ML concepts covered in the course

- Supervised Learning: Regression & Classification
- Decision Trees
- Neural Networks
- Support Vector Machines
- Clustering & Unsupervised Learning
- Evaluation Metrics
